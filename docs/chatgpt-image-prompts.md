# Промпти для ChatGPT Pro — 23 картинки

**Як це працює:**
1. Відкрийте https://chatgpt.com (Pro-план з image generation, GPT-5/4o image).
2. Для кожного блоку нижче — **новий чат**: вставте промпт, дочекайтесь зображення, натисніть «завантажити» (download).
3. Збережіть файл **рівно з тим іменем**, що вказано після промпта (наприклад `hero-tiktok.png`).
4. Перетягніть файли в відповідну теку:
   - hero-*.png → `agency-site/public/uploads/source/hero/`
   - team-*.png → `agency-site/public/uploads/source/team/`
   - site-*.png → `agency-site/public/uploads/source/sites/`
5. Скажіть мені «готово» — я запущу `npm run import:sources` і все прив'яжеться до банерів/команди/сайтів автоматично.

**Загальний стиль** (повторюється в кожному промпті): 2026 cinematic, glassmorphism,
photoreal, agency-tier, palette `#1f9fe0 #6c5ce7 #00c2a8 #ff6b9a`, no text, no logos.

## Що далі

Коли всі/частина файлів у правильних теках — кажіть «готово».
Я запущу `npm run import:sources` і:

1. Перенесу їх у `public/uploads/` з унікальними іменами
2. Створю записи в `MediaAsset`
3. Прив'яжу до `HeroBanner.imageUrl`, `TeamMember.photoUrl`, оновлю `SitesGrid` на DB-приводний компонент

Можна заливати не всі одразу — імпорт ідемпотентний: пропустить уже
обробленi файли, додасть нові.
